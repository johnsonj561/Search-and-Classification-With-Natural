Text Classification with Naive Bayes

The above interface uses a JavaScript natural language processing library, 'Natural', to perform text classification. A Naive Bayes classifier was trained against The 4 Universities Data Set. The original data set contains web pages from computer science departments in 1997. The pages were labelled into 7 categories:

  • student
  • faculty
  • staff
  • department
  • course
  • project
  • other

For this project, a simplified data set which has been stemmed and split into training (2803 docs) and test (1396 docs) sets is used for the training of the classification model. Thesimplified data set includes 4 categories:

  • student
  • faculty
  • course
  • project
